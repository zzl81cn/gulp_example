# gulp_example
> gulp构建工具的一个示例小项目，目前包含的任务是http服务器和sass编译及添加-moz等。<br/>
结合webpack是现在更加流行的方式，后面会择机整合一下。

#### install
``` bash
npm install
```

#### run
``` bash
gulp or gulp serve
```

#### build
``` bash
gulp deploy
```

#### after build run server(only run dist)
``` bash
gulp ps
```
